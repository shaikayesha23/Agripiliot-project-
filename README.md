# 🌱 AgriPilot - AI Co-Pilot for Vertical Farming

AgriPilot is an AI-powered smart farming platform designed to improve the efficiency of vertical farming. The project helps monitor crop health, analyze environmental conditions, and support data-driven agricultural decisions using Artificial Intelligence and IoT concepts.

---

## 📌 Project Overview

AgriPilot aims to enhance sustainable farming by monitoring important parameters such as temperature, humidity, soil moisture, and light intensity. The platform provides real-time insights through an interactive dashboard, helping farmers optimize crop growth and productivity.

---

## ✨ Features

- 🌱 Smart crop health monitoring
- 🤖 AI-assisted farming insights
- 🌡️ Temperature & humidity monitoring
- 💧 Soil moisture analysis
- ☀️ Light intensity tracking
